# BYOK AI Shell Android

这是手机优先的 Android 版本，和 `kotlin-desktop/` 是两套独立工程。

## 打开方式

1. 在 Android Studio 里 `File > Open`
2. 选择 `C:\Users\30501\Desktop\GTP-FIG\android-app`
3. 等 Gradle Sync 完成
4. 连接安卓手机并开启 USB 调试
5. 选择设备后直接 `Run`

## 特点

- 本地保存会话和设置
- API Key 使用 Android 安全存储
- 直接调用 OpenAI Responses API
- 手机端交互优先
